# SmartAM_System - 智能制造增材制造监控系统

> 基于多模态融合的 FDM 打印过程监测与调控系统

---

## 项目概述

本系统是一个**前后端分离**的智能监控平台，用于 FDM 3D 打印过程的实时监测与闭环调控。

### 核心功能

| 功能 | 说明 |
|------|------|
| **多模态数据采集** | IDS随轴相机 + 罗技旁轴相机 + Fotric红外热像仪 |
| **AI 缺陷预测** | PacNet 模型实时预测打印参数偏差 |
| **闭环调控** | 基于时间序列置信度的自适应参数调控（第三章算法） |
| **远程监控** | Vue 3 Web 界面实时显示视频流和状态 |

### 闭环调控原理

```
┌──────────────────────────────────────────────────────────┐
│  闭环调控流程 (论文第三章)                                │
│                                                          │
│  预测窗口 (N 个样本)                                      │
│       ↓                                                  │
│  时间衰减置信度计算 ←── 越新样本权重越高                   │
│       ↓                                                  │
│  序列一致性判断 ←── 避免偶发误检                          │
│       ↓                                                  │
│  调控决策 ←── 置信度 > 阈值 ? 执行调控 : 继续监控          │
│       ↓                                                  │
│  调控量计算 ←── 基于偏离程度和置信度                       │
│       ↓                                                  │
│  G-code 发送 (M220/M221/M290/M104)                       │
└──────────────────────────────────────────────────────────┘
```

## 快速开始

### 1. 环境要求

- Python 3.8+
- Node.js 16+
- CUDA 11.0+ (GPU 推理)
- Windows 10/11 或 Linux

### 2. 安装依赖

**后端:**
```bash
cd SmartAM_System/backend
pip install -r requirements.txt
```

**前端:**
```bash
cd SmartAM_System/frontend
npm install
```

### 3. 启动服务

**启动后端:**
```bash
cd SmartAM_System/backend
python main.py
```

**启动前端:**
```bash
cd SmartAM_System/frontend
npm run dev
```

服务启动后访问：
- **前端界面**: http://localhost:5173
- **API 文档**: http://localhost:8000/docs
- **视频流**: http://localhost:8000/video_feed

## 前端特性

### 页面功能

| 页面 | 功能描述 |
|------|----------|
| **Dashboard** | 实时监控：视频流、PacNet预测、温度曲线、闭环控制 |
| **Analysis** | 数据分析：预测历史曲线、调控记录、数据导出 |
| **Control** | 系统控制：打印机控制、PID调参、闭环调试 |
| **Settings** | 系统设置：API连接测试、相机配置、模型配置 |

### 技术亮点

- **实时视频流**: MJPEG 流，支持 IDS/旁轴/热成像三画面
- **WebSocket 数据**: 10Hz 实时传感器数据推送
- **响应式设计**: 适配桌面、平板、手机
- **主题切换**: 深色/浅色主题 + 国际化 (中/英)
- **性能优化**: 虚拟滚动、懒加载、骨架屏

## 关键特性

### 视频流布局

浏览器访问 `/video_feed` 查看组合画面：

```
┌─────────────────┬─────────────────┐
│   IDS相机       │   旁轴相机      │
│   (随轴视角)    │   (整体视角)    │
├─────────────────┴─────────────────┤
│    红外热像图   │ PacNet 预测结果 │
│                 │ Flow: Normal    │
│                 │ Feed: High      │
│                 │ Z: Normal       │
│                 │ Hot: Low        │
└───────────────────────────────────┘
```

### PacNet 模型

- **输入**: 4模态（IDS + 旁轴 + 伪彩色热像 + 灰度热像 + 工艺参数）
- **输出**: 4任务分类（流量/速度/Z偏移/温度 → Low/Normal/High）
- **推理频率**: 5Hz（每 200ms）
