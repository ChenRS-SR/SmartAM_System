# SmartAM_System - 智能制造增材制造监控系统

> 支持 FDM / SLM / SLS 三种增材制造工艺的多设备监控平台

---

## 项目概述

本系统是一个**前后端分离**的智能监控平台，支持多种增材制造工艺的实时监测与闭环调控。

### 支持的工艺类型

| 工艺 | 全称 | 设备配置 | 状态 |
|------|------|----------|------|
| **FDM** | 熔融沉积成型 | IDS随轴相机 + 罗技旁轴相机 + Fotric红外热像仪 + OctoPrint | ✅ 完成 |
| **SLM** | 选择性激光熔化 | 双通道工业相机 + 振动传感器 + 视频诊断 | ✅ 完成 |
| **SLS** | 选择性激光烧结 | Fotric 628CH热像仪 + WTVB02-485振动传感器 + ROBOIDE舵机 | ✅ 完成 |

### 核心功能

| 功能 | 说明 |
|------|------|
| **多模态数据采集** | 多相机 + 温度 + 振动 + 位置同步采集 |
| **AI 缺陷预测** | PacNet 模型实时预测打印参数偏差 |
| **闭环调控** | 基于时间序列置信度的自适应参数调控 |
| **多工艺支持** | FDM/SLM/SLS 统一平台管理 |
| **远程监控** | Vue 3 Web 界面实时显示视频流和状态 |

## 快速开始

### 1. 环境要求

- Python 3.8+
- Node.js 16+
- CUDA 11.0+ (GPU 推理，可选)
- Windows 10/11 或 Linux

### 2. 安装依赖

**后端:**
```bash
cd SmartAM_System/backend
pip install -r requirements.txt
```

**前端:**
```bash
cd SmartAM_System/frontend
npm install
```

### 3. 启动服务

**启动后端:**
```bash
cd SmartAM_System/backend
python main.py
```

**启动前端:**
```bash
cd SmartAM_System/frontend
npm run dev
```

服务启动后访问：
- **前端界面**: http://localhost:5173
- **API 文档**: http://localhost:8000/docs
- **视频流**: http://localhost:8000/video_feed

## 工艺特性

### FDM (熔融沉积成型)

```
┌─────────────────────────────────────────┐
│  监控参数                                │
│  • IDS随轴相机 (实时表面质量)            │
│  • 罗技旁轴相机 (整体视角)               │
│  • Fotric热像仪 (层间温度)               │
│  • 打印头坐标 (M114)                     │
│  • 温度/流量/速度参数                    │
├─────────────────────────────────────────┤
│  闭环调控                                │
│  • M220 速度调节                         │
│  • M221 流量调节                         │
│  • M290 Z偏移微调                        │
│  • M104 温度调节                         │
└─────────────────────────────────────────┘
```

### SLM (选择性激光熔化)

```
┌─────────────────────────────────────────┐
│  监控参数                                │
│  • 双通道工业相机 (CH1/CH2)              │
│  • 振动传感器 (铺粉检测)                 │
│  • 热成像温度监控                        │
│  • 激光功率/扫描速度                     │
├─────────────────────────────────────────┤
│  特色功能                                │
│  • 振动触发图像采集                      │
│  • 视频诊断与回放                        │
│  • 实时健康状态监测                      │
│  • 闭环参数调控                          │
└─────────────────────────────────────────┘
```

### SLS (选择性激光烧结)

```
┌─────────────────────────────────────────┐
│  监控参数                                │
│  • Fotric 628CH 热成像 (HTTP API)        │
│  • WTVB02-485 振动传感器 (Modbus RTU)    │
│  • ROBOIDE 舵机 (铺粉控制)               │
│  • 温度/振动/位置同步                    │
├─────────────────────────────────────────┤
│  特色功能                                │
│  • 7种振动检测算法                       │
│    - composite (综合优化)                │
│    - velocity_based (速度优先)           │
│    - displacement_based (位移优先)       │
│    - frequency_based (频率优先)          │
│    - rms (均方根值)                      │
│    - peak (峰值检测)                     │
│    - energy (能量计算)                   │
│  • 铺粉动作自动检测                      │
│  • 舵机扫描控制                          │
│  • 多灵敏度级别 (1-5)                    │
└─────────────────────────────────────────┘
```

## 前端特性

### 页面功能

| 页面 | FDM | SLM | SLS |
|------|-----|-----|-----|
| **Dashboard** | 实时监控、PacNet预测、温度曲线 | 双相机监控、振动波形、健康状态 | 热成像、振动监测、舵机控制 |
| **Analysis** | 预测历史、调控记录、数据导出 | 视频诊断、采集记录、趋势分析 | 波形分析、触发记录、统计分析 |
| **Control** | 打印机控制、PID调参 | ROI配置、调控参数 | 舵机控制、振动阈值、算法选择 |
| **Settings** | API连接、相机配置 | 设备连接、COM口设置 | 设备IP、串口配置 |

### 技术亮点

- **实时视频流**: MJPEG 流，支持多画面布局
- **WebSocket 数据**: 10Hz 实时传感器数据推送
- **响应式设计**: 适配桌面、平板、手机
- **主题切换**: 深色/浅色主题 + 国际化 (中/英)
- **工艺切换**: 统一的FDM/SLM/SLS切换面板

## 关键特性

### 视频流布局

浏览器访问 `/video_feed` 查看组合画面：

**FDM模式:**
```
┌─────────────────┬─────────────────┐
│   IDS相机       │   旁轴相机      │
│   (随轴视角)    │   (整体视角)    │
├─────────────────┴─────────────────┤
│    红外热像图   │ PacNet 预测结果 │
└───────────────────────────────────┘
```

**SLM模式:**
```
┌─────────────────┬─────────────────┐
│   CH1 相机      │   CH2 相机      │
│   (主视角)      │   (辅助视角)    │
├─────────────────┴─────────────────┤
│    振动波形图   │   健康状态面板  │
└───────────────────────────────────┘
```

**SLS模式:**
```
┌─────────────────┬─────────────────┐
│   热成像图      │   温度分布      │
├─────────────────┼─────────────────┤
│   振动监测      │   舵机控制      │
└─────────────────┴─────────────────┘
```

### PacNet 模型

- **输入**: 4模态（IDS + 旁轴 + 伪彩色热像 + 灰度热像 + 工艺参数）
- **输出**: 4任务分类（流量/速度/Z偏移/温度 → Low/Normal/High）
- **推理频率**: 5Hz（每 200ms）
