# PacNet 模型集成说明

> 文档版本: 2026-02-14  
> 集成功能: 4模态融合参数预测 (Flow Rate / Feed Rate / Z Offset / Hotend Temp)

---

## 一、集成概述

### 模型输入（4模态）
| 模态 | 来源 | 尺寸 | 说明 |
|------|------|------|------|
| `ids` | IDS随轴相机 | 448×448 | 跟随打印头，局部纹理 |
| `computer` | 旁轴罗技摄像头 | 224×224 | 整体视角，环境感知 |
| `fotric` | 红外伪彩色图 | 224×224 | 温度可视化 |
| `thermal` | 灰度温度矩阵 | 224×224 | 原始温度数据 |
| `params` | 工艺参数 | 10维 | 坐标、温度、流量等 |

### 模型输出（4任务 × 3分类）
| 任务 | 类别 | 说明 |
|------|------|------|
| Flow Rate | Low / Normal / High | 流量百分比偏差 |
| Feed Rate | Low / Normal / High | 进给速度偏差 |
| Z Offset | Low / Normal / High | Z轴偏移偏差 |
| Hotend Temp | Low / Normal / High | 热端温度偏差 |

### 推理频率
- **默认**: 5Hz (每 200ms 推理一次)
- **可调**: 修改 `inference_interval` 参数
- **建议范围**: 3-10Hz

## 三、代码详解

### 3.1 PacNet 推理引擎

```python
from core.pacnet_inference import PacNetInference, get_inference_engine

# 创建推理引擎
engine = PacNetInference(
    model_path="path/to/model_full.pt",  # 自动检测变体
    device="cuda",                        # GPU推理
    inference_interval=0.2                # 5Hz
)

# 加载模型
engine.load_model()

# 执行推理
result = engine.inference(
    ids_frame=ids_image,           # IDS相机 (H, W, 3) BGR
    computer_frame=side_image,     # 旁轴相机 (H, W, 3) BGR
    fotric_frame=fotric_vis,       # 伪彩色热像 (H, W, 3) BGR
    thermal_matrix=thermal_data,   # 温度矩阵 (H, W)
    params=parameters              # 工艺参数 (10,)
)

# 获取结果
print(result.flow_rate_label)      # "Low" / "Normal" / "High"
print(result.flow_rate_conf)       # 0.0 - 1.0
```

## 六、常见问题

### Q1: 模型加载失败
**现象**: `模型文件不存在`  
**解决**: 检查 `model_full.pt` 路径是否正确

### Q2: GPU 内存不足
**现象**: `CUDA out of memory`  
**解决**: 
- 降低推理频率: `inference_interval=0.5` (2Hz)
- 或使用 CPU: `device="cpu"`

### Q3: 推理结果不更新
**现象**: 画面显示 "PacNet: Waiting..."  
**排查**:
1. 检查 4 个模态输入是否都有数据
2. 检查 `ids_frame`, `computer_frame`, `thermal_matrix` 是否为空
3. 查看日志: `[DAQ] 推理执行失败`

### Q4: 预测结果闪烁
**现象**: 结果在 Normal/High 之间跳变  
**解决**: 启用平滑处理
```python
result = engine.get_smoothed_result()  # 使用历史多数投票
```

```python

```
