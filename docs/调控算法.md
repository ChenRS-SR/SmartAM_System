## 3.2 时间衰减置信度算法

### 3.2.1 算法原理

越新的预测样本，其信息价值越高。使用指数衰减函数赋予不同权重。

### 3.2.2 数学公式

```
给定预测窗口 P = [p_1, p_2, ..., p_n]，其中 p_n 是最新样本

权重计算:
w_i = α^(n-i)  其中 α ∈ (0, 1) 是衰减因子

例如: α = 0.9
w_1 = 0.9^(n-1)  ← 最旧样本权重最低
w_2 = 0.9^(n-2)
...
w_n = 0.9^0 = 1  ← 最新样本权重最高

加权置信度:
C_weighted = Σ(w_i * c_i) / Σ(w_i)

其中 c_i 是第 i 个样本的置信度
```

```python
# backend/core/closed_loop_controller.py

def calculate_time_decay_confidence(self, predictions: list, decay_factor: float = 0.9) -> float:
    """
    计算时间衰减加权置信度
    
    Args:
        predictions: 预测窗口中的样本列表
                    [{label, confidence, timestamp}, ...]
        decay_factor: 衰减因子，默认0.9
        
    Returns:
        weighted_confidence: 加权后的置信度
    """
    n = len(predictions)
    if n == 0:
        return 0.0
    
    weights = []
    confidences = []
    
    for i, pred in enumerate(predictions):
        # 计算权重: 越新的样本权重越高
        weight = decay_factor ** (n - 1 - i)
        weights.append(weight)
        confidences.append(pred.confidence)
    
    # 计算加权平均
    total_weight = sum(weights)
    weighted_confidence = sum(w * c for w, c in zip(weights, confidences)) / total_weight
    
    return weighted_confidence
```

### 3.2.4 参数选择

| 衰减因子 α | 特点 | 适用场景 |
|-----------|------|----------|
| 0.8 | 快速遗忘旧样本 | 变化快的场景 |
| 0.9 | 平衡 | **推荐默认值** |
| 0.95 | 保留更多历史 | 变化慢的场景 |

## 3.3 序列一致性判断

### 3.3.1 算法原理

避免偶发误检，要求连续多个样本都显示同一类别的偏差才触发调控。

### 3.3.2 判断规则

```
给定预测窗口 P = [p_1, p_2, ..., p_n]

定义一致性阈值 K (如 K=5):
- 如果最近 K 个样本中，≥80% 显示同一偏离类别
- 则认为序列一致，可以进行调控
- 否则继续监控
```

```python
def check_sequence_consistency(self, predictions: list, consistency_threshold: int = 5) -> tuple:
    """
    检查预测序列的一致性
    
    Args:
        predictions: 预测窗口中的样本列表
        consistency_threshold: 一致性阈值，默认5
        
    Returns:
        (is_consistent, dominant_label, consistency_ratio)
        is_consistent: 是否一致
        dominant_label: 主导类别 (Low/Normal/High)
        consistency_ratio: 一致比例
    """
    if len(predictions) < consistency_threshold:
        return False, None, 0.0
    
    # 取最近的 K 个样本
    recent = predictions[-consistency_threshold:]
    
    # 统计各类别数量
    label_counts = {}
    for pred in recent:
        label = pred.label
        label_counts[label] = label_counts.get(label, 0) + 1
    
    # 找出主导类别
    dominant_label = max(label_counts, key=label_counts.get)
    dominant_count = label_counts[dominant_label]
    consistency_ratio = dominant_count / len(recent)
    
    # 判断是否一致 (80%以上)
    is_consistent = consistency_ratio >= 0.8 and dominant_label != 'Normal'
    
    return is_consistent, dominant_label, consistency_ratio
```

## 3.4 调控量计算

### 3.4.1 算法原理

根据偏离程度和置信度计算合适的调控量。

### 3.4.2 调控策略

```
对于每个参数 (流量/速度/Z偏移/温度):

1. 如果预测为 Low (偏低):
   - 流量: 增加 5-15%
   - 速度: 降低 5-10%
   - Z偏移: 降低 0.02-0.05mm
   - 温度: 增加 5-10°C

2. 如果预测为 High (偏高):
   - 流量: 降低 5-15%
   - 速度: 增加 5-10%
   - Z偏移: 增加 0.02-0.05mm
   - 温度: 降低 5-10°C

调控量 = 基础步长 × 置信度系数 × 安全因子
```

### 3.4.3 数学公式

```
调控量 Δ = Δ_base × C_weighted × η

其中:
- Δ_base: 基础调控步长
- C_weighted: 加权置信度 (0~1)
- η: 安全因子 (0.5~1.0)，防止过调
```

REGULATION_STEPS = {
    'flow_rate': {
        'Low': 10,    # 增加10%
        'High': -10   # 降低10%
    },
    'feed_rate': {
        'Low': -5,    # 降低5%
        'High': 5     # 增加5%
    },
    'z_offset': {
        'Low': -0.02,  # 降低0.02mm
        'High': 0.02   # 增加0.02mm
    },
    'hot_end': {
        'Low': 5,     # 增加5°C
        'High': -5    # 降低5°C
    }
}

def calculate_adjustment(self, parameter: str, label: str, confidence: float) -> float:
    """
    计算调控量
    
    Args:
        parameter: 参数名 (flow_rate/feed_rate/z_offset/hot_end)
        label: 预测类别 (Low/High)
        confidence: 置信度
        
    Returns:
        adjustment: 调控量
    """
    # 获取基础步长
    base_step = REGULATION_STEPS[parameter][label]
    
    # 安全因子 (防止过调)
    safety_factor = 0.8
    
    # 计算调控量
    adjustment = base_step * confidence * safety_factor
    
    return adjustment
```

class ClosedLoopController:
    def __init__(self):
        self.predictions_window = deque(maxlen=10)
        self.regulation_history = []
        self.config = {
            'decay_factor': 0.9,
            'consistency_threshold': 5,
            'confidence_threshold': 0.7,
            'safety_factor': 0.8
        }
    
    def add_prediction(self, prediction):
        """添加新的预测结果"""
        self.predictions_window.append({
            'label': prediction.label,
            'confidence': prediction.confidence,
            'timestamp': time.time()
        })
    
    def should_regulate(self) -> tuple:
        """
        判断是否应该进行调控
        
        Returns:
            (should_regulate, parameter, adjustment)
        """
        # 1. 检查窗口是否已满
        if len(self.predictions_window) < self.config['consistency_threshold']:
            return False, None, 0
        
        # 2. 对每个参数进行检查
        for parameter in ['flow_rate', 'feed_rate', 'z_offset', 'hot_end']:
            # 获取该参数的预测序列
            param_predictions = [p[parameter] for p in self.predictions_window]
            
            # 3. 计算时间衰减置信度
            weighted_conf = self.calculate_time_decay_confidence(
                param_predictions, 
                self.config['decay_factor']
            )
            
            # 4. 检查序列一致性
            is_consistent, label, ratio = self.check_sequence_consistency(
                param_predictions,
                self.config['consistency_threshold']
            )
            
            # 5. 判断是否需要调控
            if is_consistent and weighted_conf > self.config['confidence_threshold']:
                # 计算调控量
                adjustment = self.calculate_adjustment(
                    parameter, label, weighted_conf
                )
                return True, parameter, adjustment
        
        return False, None, 0
    
    def execute_regulation(self, parameter: str, adjustment: float):
        """执行调控"""
        # 发送 G-code
        gcode = self.generate_gcode(parameter, adjustment)
        self.send_gcode(gcode)
        
        # 记录历史
        self.regulation_history.append({
            'timestamp': time.time(),
            'parameter': parameter,
            'adjustment': adjustment,
            'gcode': gcode
        })
```
