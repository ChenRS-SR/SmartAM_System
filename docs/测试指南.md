### 2.3 视频流测试

浏览器访问: `http://localhost:8000/video_feed`

**检查画面:**
- [ ] 左上: IDS 相机（随轴视角）
- [ ] 右上: 罗技摄像头（整体视角）
- [ ] 底部: 红外热像图（彩色温度图）
- [ ] 右侧: PacNet 预测结果（4行参数）
- [ ] 底部: 坐标和温度信息

**预期显示:**
```
Flow Rate   : Normal (85%)  ← 绿色
Feed Rate   : Normal (82%)  ← 绿色
Z Offset    : Normal (78%)  ← 绿色
Hotend Temp : Normal (88%)  ← 绿色
```

**颜色含义:**
- 🟢 绿色 = Normal（正常）
- 🟡 黄色 = Low（偏低）
- 🔴 红色 = High（偏高）


## 第四阶段：闭环调控测试

### 4.1 启动闭环调控

```bash
# 启动闭环监控
curl -X POST "http://localhost:8000/api/control/start"
```

**检查状态:**
```bash
curl http://localhost:8000/api/control/status
```

**预期:**
- [ ] status: "monitoring"
- [ ] window_size: 从 0 开始增加

### 4.2 观察预测结果

持续观察视频流右侧的 PacNet 预测结果。

**记录数据:**
| 时间 | Flow Rate | Feed Rate | Z Offset | Hotend | 备注 |
|------|-----------|-----------|----------|--------|------|
| 0min | Normal    | Normal    | Normal   | Normal | 初始状态 |
| 5min |           |           |          |        |      |
| 10min|           |           |          |        |      |

### 4.3 人工干预测试

**手动制造偏差（测试调控反应）:**

1. **降低流量:**
   - 在 OctoPrint 中手动设置 M221 S80
   - 观察 PacNet 是否检测到 "Low"
   - 观察闭环是否自动增加流量

2. **增加速度:**
   - 在 OctoPrint 中手动设置 M220 S150
   - 观察 PacNet 是否检测到 "High"
   - 观察闭环是否自动降低速度

**预期:**
- [ ] PacNet 能在 2-3 秒内检测到偏差
- [ ] 闭环控制器在窗口满（约 10 个样本）后做出决策
- [ ] 自动发送 G-code 调整参数

### 4.4 查看调控历史
